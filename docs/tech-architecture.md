# 技术架构说明

## 一、整体架构概览

AI-WeMedia 系统采用"AI大模型 + 平台API + 自动化工作流"的三层架构，通过统一的编排层将多个 AI 大模型与内容平台串联，实现内容生产到分发的全链路自动化。

```
┌──────────────────────────────────────────────┐
│              用户交互层 / 智能体层              │
│   （需求沟通 · 选题分析 · 提示词优化智能体）     │
└───────────────────┬──────────────────────────┘
                    │
┌───────────────────▼──────────────────────────┐
│                  编排 / 工作流层                │
│  （热点收集 → 内容生成 → AI检测 → 自动发布）    │
└──────┬────────────┬────────────┬─────────────┘
       │            │            │
┌──────▼───┐  ┌─────▼────┐  ┌───▼──────────────┐
│ AI大模型层 │  │ 数据采集层│  │ 平台发布/监控层   │
└──────────┘  └──────────┘  └──────────────────┘
```

---

## 二、各层技术选型

### 2.1 AI 大模型层

| 模型 / 平台 | 主要用途 |
|------------|---------|
| **Gemini** | 选题分析、用户画像分析、账号合规检测、内容质量评估 |
| **豆包（扣子空间）** | 视频脚本生成、平台方案制定、内容主题清单生成 |
| **腾讯 ima** | 微信公众号图文内容生成（AI 比例 ≤ 70%） |
| **kimi** | 多平台用户属性特征获取、辅助选题分析 |

### 2.2 内容分发平台

| 平台 | 内容形式 | 接入方式 |
|------|---------|---------|
| 微信公众号 | 图文 | 腾讯 ima / 公众号 API |
| 小红书 | 图文 | 平台 API |
| 抖音 | 短视频 | 平台 API |
| 快手 | 短视频 | 平台 API |
| B站 | 中长视频 | 平台 API |

### 2.3 数据采集层

| 数据源类型 | 说明 |
|-----------|------|
| 平台原生 API | 微博热搜、百度指数、各平台官方数据接口 |
| 第三方热点平台 | 聚合热点新闻与趋势数据 |
| 行业标杆指南 | 行业标杆账号的内容与动态追踪 |
| 行业官网 | 行业门户网站的最新资讯 |

### 2.4 自动化发布层

- 各平台自动发布接口对接。
- 支持按内容属性（平台、时段、受众）自动匹配发布策略。
- 支持定时发布与批量发布。

### 2.5 监控与数据看板层

| 监控维度 | 工具 / 方案 |
|---------|-----------|
| 日数据监控 | 粉丝、内容、互动数据实时仪表盘 |
| 周数据监控 | 内容效果、用户行为、营销规律周报 |
| 竞品数据监控 | 3-5 个竞品数据对比追踪 |
| AI 内容检测 | AI 比例检测工具（目标 ≤ 70%） |

## 三、内容生产工作流

```
热点收集
  ├── 原生平台热点源
  ├── 第三方热点平台
  ├── 行业标杆指南
  └── 行业官网
         ↓
  热点题材匹配（大模型）
         ↓
  内容生成
  ├── 图文：腾讯 ima → 公众号 / 小红书
  └── 视频：豆包（脚本） → 分镜图片 → 视频合成 → 抖音 / 快手
         ↓
  AI 检测与优化（去 AI 味，混合 AI 复合优化）
         ↓
  自动发布（按平台策略）
```

---

## 四、安全与合规

- AI 内容比例：所有发布内容 AI 生成比例不超过 70%，通过检测工具实时校验。
- 账号合规：通过 Gemini 大模型定期对已有账号进行合规检测，制定账户评判标准池。
- 数据隐私：用户画像数据仅用于内部运营分析，不对外共享。

## 五、核心技术指标

| 指标 | 目标值 |
|------|--------|
| 内容产出效率提升 | ≥ 3x |
| AI 内容比例 | ≤ 70% |
| 热点响应时间 | ≤ 2 小时 |
| 多平台同步发布 | 支持 6+ 平台 |
| 数据监控维度 | ≥ 20 项核心指标 |
| 竞品追踪数量 | 3-5 个 |
