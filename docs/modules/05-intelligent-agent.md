# 模块五：自媒体选题分析智能体

## 模块简介

本模块是面向自媒体创作新手与学员的引导性智能体，通过系统化的需求沟通、选题分析、平台选择和方案制定，帮助用户从零建立自媒体运营体系。

---

## 核心功能

### 5.1 需求沟通

引导用户厘清以下核心问题：
- 怎么确定选题？什么方向更容易产生效果？
- 怎么确定触达用户？是否适合普通人泛运营？
- 怎么触达目标用户？
- 表单数据的填写指引与可能遇到的运营难题提示。

### 5.2 自媒体选题分析

- **信息素养评估**：制定自媒体选题方向的信息素养标准。
- **初维度指标制定**：建立评估选题方向的多维度指标体系。
- **大模型分析**：将学员信息交给 Gemini 大模型进行分析，评估是否具备自媒体选题潜力。

**回显说明**：
- 以"自媒体选题方向指标维度文档"为标准，针对"自媒体选题能力方向"提供数据跟进分析，指导学员明确自媒体造题方向。

### 5.3 自媒体平台选择

- **平台属性获取**：获取各个自媒体平台的用户属性特征（参考：kimi、豆包-扣子空间）。
- **平台匹配分析**：根据选题方向，结合各平台特性，确定最适合运营的自媒体平台组合。
- **分析报告输出**：生成平台选择分析报告，说明各平台选择依据与预期效果。

  **提示词示例**：根据多平台用户属性（用户年龄分布、活跃时间、内容偏好、互动习惯）与选题方向的关联特性，结合各平台的内容形式、流量机制等差异化特性，确定最适合运营的自媒体平台组合，并给出详细分析报告。

### 5.4 各平台基础配置方案

- **账号定位框架**：根据框架方案指标表，通过大模型为各平台生成账号定位与建设方案。
- **内容运营方案**：生成各平台的内容运营框架，包含定位方案与内容运营细节。
- **内容主题清单**：提供与选题方向相关的内容主题清单，数量不少于 200 条。

  **扣子空间提示词示例**：
  > 以"公众号自媒体运营方案框架维度指标表"文档为标准，针对选题方向"AI工具提升职场效率"，生成微信公众号账号定位和建设部分，并提供一份选题方向相关的内容主题清单，数量不少于 200 条。

### 5.5 内容视觉、生产、分发和运营方案

- 制定各自媒体平台的内容运营框架方案标准。
- 通过大模型根据框架方案指标生成完整方案，涵盖视觉设计、内容生产流程、分发策略和日常运营规范。

### 5.6 学员技能提取

- 根据内容方案，提取出学员需要学习的核心技能点。
- 整理技能培训关键点，形成学习路径建议。
